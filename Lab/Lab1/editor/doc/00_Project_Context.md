# 项目上下文：Lab1 文本编辑器

## 项目目标
实现一个基于命令行的文本编辑器，支持多文件管理、撤销重做、日志记录和工作区状态持久化。

## 技术栈
* **语言**: Java (JDK 23)
* **编码**: UTF-8
* **构建工具**: Maven  

## 核心设计原则
1.  **模块化**: 严格区分数据层、逻辑层和交互层。
2.  **设计模式**: 必须使用命令模式 (Command)、观察者模式 (Observer) 和备忘录模式 (Memento)。
3.  **数据结构**: 文本内容必须使用 `List<String>` (行数组) 存储。
